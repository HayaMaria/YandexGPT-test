# Test API

## Описание
Этот проект берёт список английских слов и формирует таблицу с этими словами, их транскрипциями русскими буквами и переводом.

## Возможности
- Генерация таблицы слов  
- Добавление транскрипции  
- Добавление перевода  

## Требования
- Python 3.x  
- API-ключ Яндекс.Клауд для доступа к переводчику
