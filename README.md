# Test API

## Описание
Программа обрабатывает список английских слов, получая их транскрипцию и перевод с помощью нейросети. Результаты записываются в текстовую таблицу, пригодную для просмотра или дальнейшей работы.

## Возможности
- Генерация таблицы слов  
- Добавление транскрипции  
- Добавление перевода 
- Обработка ошибок: слова, которые не удалось распознать или перевести, сохраняются отдельно для повторной обработки.

## Требования
- Python 3.12 
- PyJWT — для работы с токенами Yandex Cloud 
- requests — для запросов к нейросети
- API-ключ Yandex Cloud для доступа к нейросети.

## Установка
(Что-то для устанвки) 
  
- Создайте ваши собственные файлы на основе шаблонов из папки templates, убрав из их названия "TEMPLATE", чтобы получились secrets.json, words.txt, table_words.txt, failed_words.txt, token_cache.txt.

**Никогда не выкладывайте secrets.json и token_cache.txt в репозиторий.**
- Положите эти файлы в одну папку. 
- Путь к этой папке программа получает одним из способов:
 1. Переменная окружения KEY_FOLDER 
 2. Домашняя библиотека пользователя

## Заметки для пользователя по использованию 
- В secrets.json пользователь вставляет свой ключ Yandex Cloud. 

**Никогда не выкладывайте ключ в репозиторий.**
- В words.txt кладёт английские слова (по одному на строку). 
- После запуска таблица появляется в table_words.txt. 
- Если количество ошибок > 0 — проверить failed_words.txt для повторной обработки слов.
